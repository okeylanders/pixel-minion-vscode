# Features

Planned features and enhancements for Pixel Minion.

## Format

Each feature has its own file: `feature-XXX-description.md`

## Status Legend

- **Planned** - Feature is planned but work hasn't started
- **In Progress** - Currently being worked on
- **Completed** - Implemented and released (moved to archive)
- **Deferred** - Postponed to a future version
