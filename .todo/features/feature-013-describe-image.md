# Feature-013: Describe Image Button

**Status:** Deferred
**Planned Version:** v1.1
**Priority:** Medium
**Created:** 2025-11-30

## Summary

Add "Describe" button to get AI descriptions of images.

## Description

Allow users to get detailed AI descriptions of reference images or generated images, which can then be used to improve prompts.

## Use Cases

1. Describe uploaded reference image to understand what AI "sees"
2. Describe generated image to refine prompt
3. Send reference images to Enhance Prompt Assistant

## Requirements

- [ ] Add "Describe" button to Image tab
- [ ] Add "Describe" button to SVG tab
- [ ] Implement description handler using vision model
- [ ] Display description in a modal or panel
- [ ] Option to copy description to prompt
- [ ] Option to send to Enhance Assistant

## UI Changes

- Add describe action button on images
- Description display modal/panel
- Copy/use description actions
