# Feature-010: Flip Through Past SVG Generations

**Status:** Planned
**Planned Version:** TBD
**Priority:** Low
**Created:** 2025-11-30

## Summary

Navigate through past SVG generations within a thread.

## Description

Allow users to flip through previous SVG versions generated in the current thread, similar to an undo/history stack.

## Requirements

- [ ] Track all SVG versions in thread state
- [ ] Add navigation controls (prev/next arrows)
- [ ] Show version indicator (e.g., "3 of 5")
- [ ] Allow selecting any past version
- [ ] Option to branch from a past version

## UI Changes

- Add prev/next arrows to SVG preview
- Show version count/indicator
- Consider thumbnail strip for visual navigation
